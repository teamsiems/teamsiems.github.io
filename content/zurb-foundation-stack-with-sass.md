Here are the programs I installed that allowed me to create a [Foundation 5](http://foundation.zurb.com/) project using their Sass (and other) files. I'm running Windows 7 Enterprise (64 bit).

<!-- /wp:paragraph -->

<!-- wp:paragraph -->

--- These require Windows administrator rights ---

<!-- /wp:paragraph -->

<!-- wp:paragraph -->

Ruby (1.9+) for Windows (I got version 1.9.3)  
<https://www.ruby-lang.org/>  
I used <http://rubyinstaller.org/downloads/>  
Installed to C:\Ruby193\bin\ruby.exe

<!-- /wp:paragraph -->

<!-- wp:paragraph -->

Git for Windows  
<http://git-scm.com/>  
Installed to C:\Users\\user-name\]\AppData\Local\Programs\Git\cmd\git.exe

<!-- /wp:paragraph -->

<!-- wp:paragraph -->

GitHub for Windows  
<https://windows.github.com/>

<!-- /wp:paragraph -->

<!-- wp:paragraph -->

Node.js (I got version 0.10.29)  
[http://nodejs.org/  
](http://nodejs.org/)This also installs npm.  
Installed to C:\Program Files (x86)\nodejs\node.exe

<!-- /wp:paragraph -->

<!-- wp:paragraph -->

You might need to create the 'npm' folder in C:\Users\\user-name\]\AppData\Roaming\\ to be able to use npm.

<!-- /wp:paragraph -->

<!-- wp:paragraph -->

--- These require npm ---

<!-- /wp:paragraph -->

<http://yeoman.io/>

<!-- wp:paragraph -->

Grunt (I got version 0.1.13)  
<http://gruntjs.com/>  
Installed to C:\Users\\user-name\]\AppData\Roaming\npm\grunt

<!-- wp:paragraph -->

Bower (I got version 1.3.12)
