Post vs. Page

Posts have...

  • included in feeds
  • categories
  • tags
  • excerpts
  • comments and trackbacks
  • custom fields

Pages have...

  • not included in feeds
  • page parent (not categories)
  • template
  • menu order
  • comments and trackbacks
  • custom fields

Pages can...with plugins

  • included in feeds
  • categories
  • tags
